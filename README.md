# PlayStation 3 DualShock Controller: USB-C Mod

TheÂ **PlayStation 3 DualShock USB-C Kit** is a custom board that lets you upgrade your controller by replacing the original **MiniUSB** connector with a modern, universal **USB-C** connector.

Replace your old or damaged **MiniUSB** and upgrade to modern convenience. **This USB-C kit lets you charge your controller using any standard USB-C cable**.
